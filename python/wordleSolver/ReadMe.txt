Aim of this project is to solve a worlde puzzle.

First of all program provides a random word.(rondom word comes from "words.txt")

You have to type that word as a prediction to wordle.

Then you will provide the pattern. 'g' for green letters, 'o' for orange letters, and 'b' for black letters. For example: "bbgog".

Then program will provide all the possible words left. You can pick any word you want from the list, and put it as the next word.

Program goes on like that, amount of possible words will decrease as you provide new words and petterns.

Finally, you will get the correct word.
